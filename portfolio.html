Project

스터딧(Study_IT)

개발 기간 : 2020.12.01 ~ 2021.01.27

개발환경

개발언어 : Java
WAS : apache-tomcat-8.5.54
프레임워크 : Spring Framework
데이터베이스 : Oracle
그 외 : MyBatis, HTML5, css3, javascript, bootstrap
프로젝트 소개

스터딧(Study-IT)은 IT인을 위한 신뢰성있는 스터디 매칭 및 정보공유 커뮤니티 사이트 입니다. 강북 쌍용 교육센터의 풀스택 개발자 교육 과정에서 진행했던 팀프로젝트로써 단순히 기능 구현에만 집중하기 보다 해당 서비스의 정책을 수립하는 부분에서도 신경을 많이 썼습니다. 기획 단계에서 시간과 노력을 많이 할애하며 해당 부분이 개발하는데에 있어 얼마나 중요한지 몸소 깨닫게 해준 프로젝트였습니다.

맡은 부분

제가 스터딧 개발에서 담당한 부분은 개설된 스터디 조회, 자신이 참여하고 있는 스터디의 상태에 따른 상세 정보 조회, 출석 체크, 스터디 게시글 작성, 수정, 삭제, 스터디원 상호평가 외 스터디 후기 게시판 CRUD 기능입니다.
사이트의 정책을 수립하는 과정은 팀원 모두가 함께 했으며 그 이후 각자 맡은 부분의 테이블 설계, 쿼리문 작성, view(html, css, bootstrap, js), 자바단 코딩은 따로 처리하고 하나로 합치는 과정을 거쳤습니다.
